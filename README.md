# Haustier-App

Ein simples digitales Haustier als Schulprojekt.

## Funktionen
- Haustier auswählen
- Hunger, Energie, Laune und Sauberkeit anzeigen
- Buttons zum Füttern, Spielen, Schlafen und Waschen
- Werte ändern sich durch Klicks
- Zustandstext reagiert automatisch
- Supabase speichert den Haustier-Zustand

## Dateien
- `index.html`
- `style.css`
- `script.js`

## Supabase
Die App nutzt die Tabelle `pet_state` im Projekt `haustierapp`.
